# Project Portfolio 1: kaleidoscope-learning
This website has been created to fulfil the project brief for Code Institute's first individual assessment.
The live link can be found here: https://gilberto2025123.github.io/kaleidoscope-learning/#home


## Description
Kaleidoscope Learning is a fully responsive static HTML and CSS website designed to inform and provide resources for those engaging in diversity and inclusion topics within educational spaces.

## User Stories
1. User Story:
As a student, I want to quickly understand diversity expectations so I can prepare for my future career and contribute positively to inclusive environments.

      Acceptance Criteria:
      •Hero section immediately explains why D&I matters
      • Content uses accessible language without jargon
      • Mobile-responsive design for smartphone access
      • Clear navigation between sections
      Key Tasks:
      • Create Hero image
      • Write About Intro with highlighting D&I principles
      • Add a card with an image

2. User Story: 
As an educator, I want practical guidance on creating inclusive classrooms so I can support all students effectively and model best practices. 

     Acceptance Criteria:
     • Information sections address classroom management and curriculum design 
     • Tips list includes actionable classroom strategies  
     • Resource links connect to educational D&I frameworks 
     Key Tasks:
     • Design grid layout showcasing classroom inclusion principles 
     • Create Bootstrap list group with teaching-specific tips  
     • Add external links to educational D&I resources 
     • Include images representing diverse learning environments 

3. User Story:
As an institutional decision-maker, I want comprehensive D&I implementation guidance so I can develop effective policies and training programs. A

      Acceptance Criteria:
      • Event section includes CTA to go to the contact section
      • Content addresses both student and staff diversity
      Key Tasks:
      • Implement Bootstrap carousel components for events
      • Design professional colour scheme and typography
      • Add institutional partnership examples

4. User Story:
As a D&I trainer, I want current best practices and resources so I can enhance my training programs and stay updated on industry standards.
 
   Acceptance Criteria:
    • Contact form
    Key Tasks:
    • Include a contact/networking opportunities section
    • Message text area box
    • Submit message button 

5. User Story:
As a policy advocate, I want evidence-based D&I information so I can support legislative initiatives and community programs effectively.

     Acceptance Criteria:
     • Resource section links to government guidelines and links
     • Information supports policy development discussions
     Key Tasks:
     • Design subscription modal section using Bootstrap components
     • Implement accessibility features (WCAG compliance)
     • Add government partnership and contact information links in the footer
         

6. User Story:
As a student, I want to quickly understand diversity expectations so I can prepare for my future career and contribute positively to inclusive environments.

      Acceptance Criteria:
      • Video showcasing diversity and inclusion principles in education.
      • Must be informative for both students and professionals
      Tasks:
      • Source video from YouTube link to explainer
      • Link to resources section
      • Add to the resources section.

7. User Story:
 As a visitor to this website, I should be informed and attracted by the easy accessible navigation of the website so that I can learn more about diversity and inclusion.

     Acceptance Criteria:
    • Make a nav bar with sections that are fully responsive across devices
     • Ensure colours are suitable for accessibility purposes.
     Tasks:
     • Ensure to test accessibility with Lighthouse and ami-responsive tools
     • Design a connected and responsive page

## Typography and Colour Scheme
### Colour Scheme
This colour palette was chosen to provide a bright and calming space to ensure accessibility for all. These colours are vibrant to reflect the spectrum of diversity and to bring about a celebratory note.

### Typography
These fonts were used to provide an attractive and playful interface but one that is also accessible to all. Fitting in with the project brief. 

Bariecito- Used for headings and section intros 

Nunito Sans - Used for the main body of the text and can be viewed easily across devices.

## Features
There were a few main features included in this one page project. Listed here in no chronological order: 
* Navigation Bar, with links for the sections on the page(About, Resources, Events, Contact). Fixed top
* A Footer with external links / Social media icons.
* A contact form
* A carousel image slider
* A grid card group
* CTA buttons like "Send Message", "Learn More"
* Scroll page
* Images
